# Configs

This folder contains the full raw configuration files for the lab devices. These are posted as-is, so anyone wanting to skim the current config  can do so without digging through a guide.

Filenames indicate the device they belong to.

- **Why things are configured this way** — see the [`docs`](../docs) folder for the design decisions and reasoning behind these configs.
- **How to get to this final config** — see the [`labs`](../labs) folder for step-by-step guides showing the process of building up to the config posted here.

More raw configs will be added here over time as new devices are set up.
